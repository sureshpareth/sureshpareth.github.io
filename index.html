Teaching Expertise

Artificial Neural Networks

Artificial Neural Networks (ANNs) have emerged as a cornerstone of modern machine learning and artificial intelligence. In this comprehensive course, we will delve into the core concepts, methodologies, and applications of ANNs, exploring their power in nonlinear modeling and their connection to kernel spaces. Convolutional Neural Networks (CNNs), a specialized form of ANNs, will take center stage. We will explore their unique architecture, which has revolutionized image and visual data processing. Understanding CNNs is pivotal for tasks such as image recognition, object detection, and image generation. Moving forward, we will delve into Recurrent Neural Networks (RNNs), which excel at sequential data processing and have led to significant advancements in natural language understanding, machine translation, and speech recognition. We will explore the concept of memory in RNNs and its implications for processing time-series data.

Foundations of Deep Learning

The essence of Deep Learning, showcasing its transformative power at the intersection of artificial intelligence and computer science. Just as a masterful tapestry weaves together diverse threads, Deep Learning intertwines neural networks, hierarchical representations, and advanced architectures to unravel complex patterns from data. This evolving discipline has not only revolutionized domains like computer vision, speech recognition, and natural language processing but also holds the promise to emulate cognitive processes, fueling innovation across fields like healthcare, agriculture, and beyond. Just as Machine Learning is a beacon of innovation, Deep Learning stands as a beacon of intricacy, with its deep architectures and intricate algorithms shaping the future of artificial intelligence and problem-solving on an unprecedented scale.

Machine Learning

This course provides a foundational yet comprehensive introduction to Machine Learning, exploring its basic principles, diverse applications, and inherent challenges. The curriculum is divided into key areas, starting with Supervised Learning, where students will master techniques such as Linear Regression (single and multiple variables), Gradient Descent, Classification, Logistic Regression, and methods to combat Overfitting through Regularization. Advanced supervised models like Support Vector Machines, Artificial Neural Networks (including Perceptrons, Multilayer Networks, and back-propagation), Decision Trees, and Ensemble Methods will also be covered. The course then transitions to Unsupervised Learning, focusing on Clustering algorithms like K-means, K-medoids, and Hierarchical Clustering. Students will also learn about crucial Dimensionality Reduction techniques, particularly Principal Component Analysis (PCA). Throughout the course, emphasis will be placed on understanding the practical applications of various machine learning methods across different domains, equipping students with the knowledge to implement and evaluate these powerful algorithms.

Fundamentals of Computer Systems

This course, aligned with Computer Systems: A Programmer's Perspective by Randal E. Bryant and David R. O'Hallaron (Pearson, 2016), introduces the core concepts of computer systems from a programmer's viewpoint. It begins with an introduction to computer systems, exploring how hardware and software interact. Program structure and execution are covered, focusing on how information is represented and manipulated, including binary data, integers, and floating-point numbers. The machine-level representation of programs is examined, teaching how high-level code translates to assembly and machine instructions. Processor architecture is explored, detailing how CPUs execute instructions and process data. Techniques for optimizing program performance are introduced, emphasizing efficient code and resource use. The memory hierarchy is studied, covering caches, main memory, and storage to understand speed and efficiency trade-offs. Linking is explained, detailing how object code is combined into executable programs. Exceptional control flow is addressed, including interrupts, exceptions, and process control. Virtual memory is covered, explaining how systems manage memory allocation and protection. The fundamentals of operating systems are introduced, highlighting process management, file systems, and concurrency. This course, grounded in Bryant and O'Hallaron’s text, equips students with a deep understanding of how programs run on modern computer systems.

Python Programming Lab

This course provides a comprehensive introduction to Python programming, designed for beginners and those looking to solidify their foundational knowledge. Students will learn the essentials of writing, testing, and debugging Python programs, mastering core control flow structures like loops and conditionals. Python Programming

This course offers a comprehensive introduction to programming, focusing on Python as a versatile and beginner-friendly language. It starts with an overview of programming languages, highlighting their purpose and types. Python basics are covered, including syntax, setup, and writing simple programs. Students learn about variables and data types, such as integers, floats, strings, and booleans, to store and manipulate data. Control structures, including conditionals like if-else statements, are explored to manage program flow. Repetition structures, such as for and while loops, are introduced to handle iterative tasks. The course delves into functions and modules, teaching how to define reusable code blocks and organize programs using Python’s modular system. String manipulation and list operations are covered for handling text and collections of data. File input and output techniques are taught for reading from and writing to files. Basic data structures, like tuples, sets, and dictionaries, are introduced for efficient data organization. Object-oriented programming concepts, including classes, objects, inheritance, and encapsulation, are explained to build structured and scalable code. Finally, the course touches on Python web frameworks, such as Django and Flask, which simplify web development by providing tools for routing, templating, and database integration to create dynamic web applications.

Database Systems

This course provides a comprehensive exploration of database systems, beginning with basic concepts to understand the purpose and principles of databases. It delves into data models and languages, covering structures like relational and hierarchical models, alongside query languages such as SQL. The course addresses database design, teaching both conceptual techniques, like creating ER diagrams, and physical design, focusing on storage and indexing. Students learn system implementation techniques to build and optimize databases effectively. Current trends in database technology are examined to keep pace with modern advancements. The course also covers distributed databases, exploring the challenges of managing data across multiple locations. Practical skills are developed through the design and implementation of database systems or packages tailored for applications like office automation, hotel management, and hospital management. This includes the deployment of forms and reports for user interaction. Normalization techniques are taught to eliminate redundancy and ensure data integrity, while query processing algorithms are studied for efficient execution within these application projects. Hands-on implementation of key components rounds out the course, preparing students for real-world database scenarios.

Optimization Techniques in AI

This course provides a comprehensive introduction to optimization techniques, with a strong focus on their applications, particularly in machine learning and multimedia processing. Students will delve into foundational concepts such as convex sets, convex functions, and Lagrange duality, building a solid theoretical understanding. The curriculum covers a wide array of optimization algorithms, including convex optimization algorithms, second-order cone models, semidefinite programming, semi-infinite programming, minimax, sublinear algorithms, interior point methods, active set methods, stochastic gradient descent, coordinate descent, and cutting planes methods. Through this course, students will gain practical skills in applying these advanced optimization tools to solve real-world problems in various computational domains.

Research Areas

Inverse and Ill-Posed Problems

Inverse problems constitute a fascinating realm where the objective is to discern the causal factors behind a given observation or effect. This field has been a subject of study for nearly a century, predominantly driven by practical applications. The scope of applications encompasses signal restoration, tomography in medical imaging, microscopy, atmospheric imaging, biometrics, and computer vision methodologies.

Machine Learning

Positioned at the intersection of artificial intelligence and computer science, Machine Learning harnesses the power of data. It finds kinship with statistics and mathematics, addressing diverse challenges spanning DNA analysis, medical diagnosis, product recommendations, stock trading, and fraud detection. The domain focuses on refining algorithms, distributed learning, and emulating natural cognitive processes.

Data Science

A multidisciplinary field dedicated to collection, preparation, analysis, visualization, management, and preservation of extensive information repositories. Data science encompasses statistics, computing technology, and artificial intelligence, driving transformative research across environmental concerns, social dynamics, healthcare, and cloud computing.

IT151 Python Programming

Course Overview

Course Code: IT151 | Credits: 3-0-0 (3 Credits)

Duration: 15 Weeks | Weekly Hours: 3 Hours | Total: 45 Hours

Textbooks: Martin C. Brown (2018), Mark Summerfield (2018)

Week 1
3 Hours
Introduction to Programming Languages & Python Basics
Overview of programming languages, history of Python, Python installation, IDLE, interactive mode, basic syntax

Learning Objectives:

  • Understand the role of programming languages
  • Install and configure Python development environment
  • Write and execute basic Python programs
  • Use Python's interactive mode effectively
Lab Activity: Setting up Python environment, writing "Hello World" program
Brown Ch. 1-2, Summerfield Ch. 1
Week 2
3 Hours
Variables and Data Types
Variable declaration, naming conventions, numeric types (int, float, complex), strings, booleans, type conversion
Prerequisites: Week 1 - Python basics

Learning Objectives:

  • Declare and initialize variables following Python conventions
  • Understand different data types and their uses
  • Perform type conversion between different data types
  • Use Python's dynamic typing effectively
Lab Activity: Variable manipulation, type checking, and conversion exercises
Brown Ch. 3, Summerfield Ch. 2
Week 3
3 Hours
Control Structures - Conditional Statements
if, elif, else statements, comparison operators, logical operators, nested conditions, boolean expressions
Prerequisites: Variables and data types

Learning Objectives:

  • Write conditional statements using if-elif-else
  • Use comparison and logical operators effectively
  • Create complex boolean expressions
  • Implement nested conditional structures
Lab Activity: Grade calculator, number classification programs
Brown Ch. 4, Summerfield Ch. 3
Week 4
3 Hours
Repetition Structures - Loops
for loops, while loops, range() function, break and continue statements, nested loops, loop control
Prerequisites: Control structures

Learning Objectives:

  • Implement for and while loops appropriately
  • Use range() function for iteration
  • Control loop execution with break and continue
  • Design nested loop structures
Lab Activity: Pattern printing, sum calculations, multiplication tables
Brown Ch. 5, Summerfield Ch. 3
Week 5
3 Hours
Functions - Part 1
Function definition and calling, parameters and arguments, return values, local vs global scope
Prerequisites: Basic programming constructs

Learning Objectives:

  • Define and call functions with parameters
  • Understand function return values
  • Distinguish between local and global scope
  • Write modular code using functions
Lab Activity: Mathematical functions, utility functions
Brown Ch. 6, Summerfield Ch. 4
Week 6
3 Hours
Functions - Part 2 & Modules
Default parameters, keyword arguments, variable-length arguments, lambda functions, importing modules, creating modules
Prerequisites: Functions basics

Learning Objectives:

  • Use advanced function features (default params, *args, **kwargs)
  • Create and use lambda functions
  • Import and use built-in modules
  • Create custom modules
Lab Activity: Module creation, math and random module usage
Brown Ch. 7, Summerfield Ch. 5
Week 7
3 Hours
Strings - Part 1
String creation, indexing, slicing, string immutability, basic string methods
Prerequisites: Basic Python syntax

Learning Objectives:

  • Manipulate strings using indexing and slicing
  • Understand string immutability
  • Use basic string methods effectively
  • Perform string concatenation and repetition
Lab Activity: String manipulation exercises, palindrome checker
Brown Ch. 8, Summerfield Ch. 6

3 Hours
Strings - Part 2
Advanced string methods, string formatting, regular expressions basics, string validation
Prerequisites: Strings Part 1

Learning Objectives:

  • Use advanced string methods for text processing
  • Format strings using various techniques
  • Apply basic regular expressions
  • Validate string input
Lab Activity: Text processing, word count, string formatting
Brown Ch. 9, Summerfield Ch. 7
Week 9
3 Hours
Lists - Part 1
List creation, indexing, slicing, list methods (append, insert, remove, pop), list comprehensions
Prerequisites: Basic Python constructs

Learning Objectives:

  • Create and manipulate lists effectively
  • Use list methods for data manipulation
  • Write list comprehensions
  • Understand list mutability
Lab Activity: List operations, data sorting, filtering
Brown Ch. 10, Summerfield Ch. 8
Week 10
3 Hours
Lists - Part 2 & Advanced Data Structures
Nested lists, tuples, sets, dictionaries, choosing appropriate data structures
Prerequisites: Lists Part 1

Learning Objectives:

  • Work with nested lists and multidimensional data
  • Use tuples, sets, and dictionaries appropriately
  • Choose optimal data structures for different scenarios
  • Convert between different data structures

Week 11
3 Hours
File Input and Output
Opening and closing files, reading from files, writing to files, file modes, exception handling with files
Prerequisites: Strings and basic data structures

Learning Objectives:

  • Read data from text files
  • Write data to text files

Basic Data Structures & Algorithms

Learning Objectives:

  • Implement basic sorting algorithms
  • Analyze algorithm efficiency basics
Lab Activity: Stack calculator, search implementations, sorting comparisons

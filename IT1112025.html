Course Plan: IT111 - Fundamentals of Computer Systems

Course Information

Instructor: Dr. Suresan Pareth

Department: Information Technology, NITK Surathkal

Term: August - December 2024

Credits: 4 (4-0-0)

Pre-requisite: C Programming

Lecture + Lab Hours: 4 Theory + 2 Lab/week

Course Description

This course provides foundational knowledge in computer systems, covering hardware-software interaction, memory, processors, program execution, and system-level optimization with real-world relevance to AI and performance computing.

Week 10Day 19Final ReviewRecap of all key concepts.

Course Outcomes

  • CO1: Understand program execution, data representation, and system architecture.
  • CO2: Analyze/debug machine-level programs.
  • CO3: Apply optimization techniques using processor and memory knowledge.
  • CO4: Evaluate ethical and sustainable aspects in systems programming.

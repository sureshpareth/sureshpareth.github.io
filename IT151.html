Course Description

This course provides a comprehensive introduction to Python programming, one of the most versatile and widely-used languages in industry and research. Students begin with the fundamental building blocks of programming — variables, data types, and control structures — and progressively advance through functions, modules, strings, and collections.

The course covers essential data structures including lists, tuples, sets, and dictionaries, along with file I/O operations and basic algorithms such as searching and sorting. A significant portion is devoted to Object-Oriented Programming (OOP) principles including classes, inheritance, polymorphism, and encapsulation.

Students gain hands-on experience through weekly lab activities, and the course concludes with an introduction to web frameworks (Flask/Django) and a final project that integrates all learned concepts. The course emphasizes clean code, modular design, and Pythonic best practices.
02

Course Educational Objectives

Python Foundations

  • Understand Python's role in modern software development
  • Set up and use the Python development environment
  • Master syntax, variables, data types, and basic I/O

Control Flow & Functions

  • Implement conditional statements and iteration constructs
  • Design modular programs using functions and modules
  • Apply advanced function features: lambdas, *args, **kwargs

Data Structures

  • Manipulate strings, lists, tuples, sets, and dictionaries
  • Choose appropriate data structures for real-world problems
  • Write list comprehensions and functional transformations

OOP Principles

  • Design classes, objects, and class hierarchies
  • Implement inheritance, polymorphism, and encapsulation
  • Use special methods for Pythonic class design

Applied Python

  • Perform file I/O and process structured data files
  • Implement basic data structures and algorithms
  • Build simple web applications using Python frameworks
03

Course Outcomes

CO1

Apply Python syntax, data types, control structures, and functions to write correct, efficient, and well-structured programs for a variety of computational problems.

CO2

Select and use appropriate Python data structures — including strings, lists, tuples, sets, and dictionaries — to store, manipulate, and process data effectively.

CO3

Design and implement object-oriented solutions in Python by applying the principles of encapsulation, inheritance, and polymorphism to model real-world problems.

CO4

Develop complete Python applications integrating file I/O, basic algorithms, and external modules, demonstrating software engineering practices including modularity and documentation.

Course Description

This course introduces students to the principles and practices of parallel computing, emphasizing the architectural, algorithmic, and programming foundations required to design efficient and scalable parallel applications. The course begins by examining the limitations of sequential execution models and motivates the need for parallelism through an exploration of modern computer architectures, including multicore and manycore processors, memory hierarchy, cache behavior, and instruction-level parallelism.

Students are introduced to fundamental parallel programming concepts such as data parallelism, task parallelism, and pipeline parallelism, along with shared-memory and distributed-memory execution models. The course provides hands-on exposure to industry-standard parallel programming frameworks, including OpenMP for shared-memory systems, MPI for distributed-memory systems, and task-parallel runtimes such as Intel TBB and Cilk++.

The course further explores heterogeneous computing environments covering GPU acceleration using CUDA and OpenCL, manycore platforms such as Xeon Phi and FPGAs, and the application of parallel computing in deep learning, scientific simulations, and large-scale data processing.
02

Course Educational Objectives

Architecture Foundations

  • Limitations of sequential execution and the need for parallelism
  • Multicore/manycore architectures, processor organization, cache memory
  • Analyze performance bottlenecks in memory hierarchy and execution models

Parallel Programming Models

  • Data parallelism, task parallelism, and pipeline parallelism
  • Shared memory and distributed memory programming paradigms
  • SPMD and MPMD execution models in parallel program design

Programming Frameworks

  • Shared-memory programming using OpenMP directives
  • Distributed-memory applications using MPI communication primitives
  • Task-parallel programming with Intel TBB and Cilk++

Heterogeneous Computing

  • Accelerator-based computing using GPUs, CUDA, and OpenCL
  • Systems involving CPUs, GPUs, Xeon Phi, and FPGAs
  • Optimization techniques balancing computation, communication, and memory

Real-World Applications

  • Parallel computing in deep learning and scientific simulations
  • Data, model, and pipeline parallelism for neural network training
  • Responsible practices: correctness, profiling, ethical HPC development
03

Course Outcomes

CO1

Apply fundamental concepts of computer architecture and parallel execution, including multicore processors, memory hierarchy, and instruction-level parallelism, to understand the limitations of sequential computing.

CO2

Decompose computational problems into parallel components using appropriate parallel programming paradigms such as data parallelism, task parallelism, and pipeline parallelism.

CO3

Validate the effectiveness of parallel programming frameworks and tools such as OpenMP, MPI, and task-parallel libraries in solving computationally intensive and real-world problems.

CO4

Design and develop efficient parallel computing solutions by integrating parallel programming techniques, synchronization mechanisms, and performance optimization strategies while ensuring correctness, scalability, and responsible use of resources.

Optimization Techniques

IT304 (3-0-2) 4 Credits
Instructor: Dr. Suresan Pareth

Course Overview

This course provides a comprehensive introduction to optimization theory and algorithms with emphasis on convex optimization and machine learning applications. Students will learn both theoretical foundations and practical implementation of optimization techniques for real-world problems.

Learning Objectives

Theoretical Foundation

Understand fundamental concepts of optimization theory and mathematical formulations

Problem Formulation

Formulate optimization problems in various domains and applications

Algorithm Implementation

Apply and implement different optimization algorithms for practical solutions

Real-World Applications

Use optimization methods in image, video, and multimedia processing applications

Weekly Course Schedule

Week 1: Introduction to Optimization

Topics:

  • What is optimization? Types of optimization problems
  • Mathematical formulation: objective function, constraints
  • Classification: linear vs nonlinear, constrained vs unconstrained
  • Optimization in machine learning and engineering applications

Learning Outcomes:

Identify and formulate optimization problems, classify different types

Week 2: Mathematical Foundations

Topics:

  • Vector spaces and norms
  • Gradients, Hessians, and Taylor expansions
  • Optimality conditions: first-order and second-order
  • Global vs local optima

Learning Outcomes:

Apply mathematical tools for optimization analysis

Week 3: Convex Sets

Topics:

  • Definition and properties of convex sets
  • Convex combinations and convex hull
  • Important examples: hyperplanes, halfspaces, ellipsoids
  • Separation theorems

Learning Outcomes:

Identify and work with convex sets, apply convexity properties

Week 4: Convex Functions

Topics:

  • Definition and characterization of convex functions
  • First-order and second-order conditions for convexity
  • Operations preserving convexity
  • Jensen's inequality and applications

Week 5: Lagrange Duality Theory

Topics:

  • Constrained optimization problems
  • Lagrangian function and dual function
  • Weak and strong duality
  • KKT conditions

Week 6-7: Convex Optimization Algorithms

Topics:

  • Gradient descent and Newton's methods
  • Projected gradient methods
  • Proximal gradient methods
  • Accelerated gradient methods (Nesterov)

Week 8: Second-Order Cone Programming

Topics:

  • Second-order cone constraints
  • SOCP formulation and examples
  • Applications: robust optimization, portfolio optimization
  • Software tools (CVX, CVXPY)

Week 9: Semidefinite Programming

Topics:

  • Positive semidefinite matrices
  • SDP formulation and examples
  • Applications: combinatorial optimization, control theory
  • Relaxation techniques

Week 10-11: Advanced Topics

Topics:

  • Semi-infinite programming
  • Minimax optimization
  • Game theory connections
  • Applications in machine learning

Week 12-13: Modern Optimization Methods

Topics:

  • Stochastic gradient descent (SGD)
  • Adaptive methods (Adam, RMSprop)
  • Coordinate descent methods
  • Parallel implementations

Week 14: Specialized Methods

Topics:

  • Interior point methods
  • Active set methods
  • Cutting plane methods
  • Sublinear algorithms

Week 15: Applications to Multimedia Processing

Topics:

  • Image denoising and deblurring
  • Compressed sensing and sparse recovery
  • Video processing optimization
  • Real-world case studies

Week 16: Project Presentations

Topics:

  • Student project presentations
  • Course review and synthesis
  • Future directions in optimization

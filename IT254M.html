IT254 / IT254M

Web Technologies and Applications

Credits: (3-0-2) 4 | Theory: 3 hrs | Lab: 2 hrs

Course Overview

This course provides comprehensive coverage of web technologies, from fundamental concepts to advanced frameworks. Students will learn both client-side and server-side programming, explore modern web architectures, and understand emerging trends in web development. The course combines theoretical knowledge with practical hands-on experience through extensive laboratory work.

Laboratory Exercises

Lab 1-2: HTML5 & CSS3

Create responsive web pages using HTML5 semantic elements and CSS3 styling techniques including flexbox and grid layouts.

Lab 3-4: JavaScript & DOM

Implement interactive web applications using JavaScript, event handling, and DOM manipulation techniques.

Lab 5-6: Forms & Validation

Develop dynamic forms with client-side and server-side validation, including file uploads and data processing.

Lab 7-8: AJAX & jQuery

Build dynamic web applications using jQuery library and AJAX for asynchronous data exchange.

Lab 9-10: Web Services

Create and consume RESTful web services, work with JSON data, and integrate third-party APIs.

Lab 11-12: Project Work

Develop a complete web application incorporating all learned technologies and best practices.
